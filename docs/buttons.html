You will need to label your buttons or circuits, there is one button for each physical circuit on the Panel, plus Heater(s).

First it's good to understand how Jandy assigns these circuits, below you will see a table for commonly used panels, if yours is not there, it should be easy to work out.

Note: If you have configured your panel_type correctly then the configuration editor will only show the appropriate buttons.
