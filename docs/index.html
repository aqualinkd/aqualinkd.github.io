Assuming you have a RS485 link to your Jandy Control Panel.

  • If not see see Wiki - connecting RS485 to control panel

    • Quickstart

  • Install AqualinkD. 
    curl -fsSL https://install.aqualinkd.com | sudo bash -s -- latest
  • Browse to http://ip.of.aqualinkd
    •  
  • Go to aqmanager by selecting the 3 lines in top left corner
    •  
  • In aqmanager, once AqualinkD has finished self configuring, expand the "Config" section and select "edit config"
  • Check your panel type is correct.
    • If you change the panel type you must hit save and reload config before changing anything else
  • Go down to the button section and configure your circuits / buttons
  • Check / change and other values and select "Save" (even if you didn't change anything)
