Assuming you have a RS485 link to your Jandy Control Panel.

  • If not see see Connecting to Panel or Wiki - connecting RS485 to control panel

    • Quickstart

  • Install AqualinkD. 
    curl -fsSL https://install.aqualinkd.com | sudo bash -s -- latest
  • Browse to http://ip.of.aqualinkd
    •  
  • Go to aqmanager by selecting the 3 lines in top left corner
    •  
  • In aqmanager, once AqualinkD has finished self configuring, expand the "Config" section and select "edit config"
