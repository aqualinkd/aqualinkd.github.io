To set RPM,GMP and read accurate RPM, GPM, Watts, Pressure Curve and many other parameters from your VSP, you will need to configure it. Each pump is assigned to a Button or Virtual Button.

The configuration options are as follows. (Where ?? is the number ie 01, and replace button_?? with virtual_button_?? if using virtual button)
  • button_??_pumpID The RS485 ID of the VSP. Running serial logger will give you this information if you don't know.
    • Pentair ID's start at 0x60
    • Jandy ID's start at 0x78
  • button_??_pumpIndex If your panel rev is prior to Yg, then you can get this number from the pump information shown your Jandy display ie Intelliflow VS 1 (ignore if you panel rev is Yg is newer)
  • Button_??_pumpName If your panel rev is Yg or newer, then this is the name of your pump. ie (ignore if panel rev is older than Yg)
  • button_??_pumpTypeThe Type of pump (JANDY ePUMP, Pentair VF, Pentair VS)
