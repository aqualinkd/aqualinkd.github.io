If you have bought the AqualinkD complete Radxa kit, this comes with everything you need pre built & installed, and ready to go. You will however need to connect it to your WiFi!
Since this is a complete Linux computer, you will need some way to get your WiFi information on the device, this can be done in one of three ways :-

  • Connect a USB drive with WiFi information to the Radxa
    • -or-
  • Connect a screen and keyboard to the Radxa
    • -or-
  • Connect a USB 2 Ethernet adapter to the Radxa and plug it into one of your routers/switch ethernet port

WARNING during these steps you may be using different cables & adapter is is VERY important to

Never connect the Radxa to both USB power and RS485 power at the same time

 

Connect a USB drive with wifi information

You will need the following
  • USB stick or equiv (ie CF card & CF card reader)
  • USB-C to USB adapter cable (if above is not USB-C)
  • optional - USB-C power source

AqualinkD complete Radxa kit comes with software pre-installed that will detect a USB drive when it's inserted, mount it look for WiFi configuration files, and if found create a WiFi connection.

Steps are as follows
  1. On a computer, format your USB stick in exFAT or FAT32.
  2. create a plane text file on the USB stick called nmcli.conf (if using windows make sure .txt is not added to the file name)
    • Preferable download the file from here (add link)
    • Contents of this file should be as below, changing the values appropiatly for your specific wifi settings

      • SSID="YOUR_SSID" PASSWORD="YOUR_PASSWORD"

    • It is important to only change the text inside the quotes YOUR_SSID and YOUR_PASSWORD
  3. Eject the USB stick from your computer
  4. Remove the Radxa from the case by removing the 4 screws on the underside of the case, and lifting it out holding the case top
  5. Connect Radxa box to power. and wait a minute for it to fully boot.
  6. Insert your USB stick into one of the Radxa USB-C ports, and wait 10~20 seconds
  7. Remove the USB stick
If everything worked, your Radxa will not be connected to your wifi network.

The output of the wifi connection will be written to a file called auto-wifi-connect.log on the USB stick,
you can view that file for the IP address your router gave out, or go to your routers DHCP page to view the IP.


http://ip.address/ or if your device supports mDNS you can use http://aqualinkd.local
 

Connect a screen and keyboard

You will need the following
  • USB Keyboard (USB-C or USB-C adapter)
  • HDMI Monitor, cable for Micro HDMI(Type-D) to Monitor
  Steps are same as any Linux computer.
  1. Remove Radxa from case, remove 4 screws for case bottom, lift up on case top to remove Radxa & HAT
  2. Connect Screen and keyboard to Radxa
  3. Power up Radxa (either from USB-C port or RS485 connector
  4. Login to machine (Default user/password is xxxxx yyyyy
  5. Run nmcli to list WiFi connection, then again to connect
    • sudo nmcli radio wifi on
    • sudo nmcli device wifi list
      • Make sure your WiFi network is listed.
    • sudo nmcli device wifi connect "YOUR_SSID" password "YOUR_PASSWORD"

Connect a USB 2 Ethernet adapter and plug it into your network

You will need the following
  • USB to Ethernet adapter
  • Ethernet cable
  • Computer with a SSH tool
  1. Remove Radxa from case, remove 4 screws for case bottom, lift up on case top to remove Radxa & HAT
  2. Connect the USB to Ethernet adapter to the Radxa
  3. Power up Radxa (either from USB-C port or RS485 connector
  4. Find the IP address of the radxa from your router
  5. Use your SSH tool to SSH to the Radxa IP, login with user / password
  6. Follow the 
    nmcli
    commands listed in Connect a screen and keyboard to the Radxa
