Virtual Buttons are similar to buttons in that they create a device on the web UI and in Home Automation hubs but the difference is there is no physical circuit defined to the Button.
They are used for Variable Speed Pumps that are not assigned to a circuit, HeatPumps/Chillers, OneTouch buttons etc.
Configuration options are similar to Standard Buttons, list is below.
  • virtual_button_??_label (Mandatory) The Label for the button
  • virtual_button_??_lightmode (optional) See Lights
  • virtual_button_??_pumpIndex (optional) See Variable Speed Pumps
  • virtual_button_??_pumpName (optional) See Variable Speed Pumps
  • virtual_button_??_pumpType (optional) See Variable Speed Pumps
  • virtual_button_??_onetouchID (optional) The ID if a OneTouch button (1 to 6)
  • virtual_button_??_altLabel (optional) If the name changes depending on mode, like a HeatPump/Chiller. Example
    • virtual_button_01_label=Heat Pump
    • virtual_button_01_altLabel=Chiller
Some examples are below.

virtual_button_01_onetouchID = 3

virtual_button_03_onetouchID = 1

virtual_button_04_altlabel = Chiller
