Getting started with the Raspberry Pi Compute Module complete kit

If you have bought the AqualinkD Raspberry Pi Compute Module kit: This comes with everything you need pre built & installed, and is ready to go. You will however need to connect it to your WiFi!

Since this is a complete Linux computer, there are a few steps to complete

  • Connect to your Jandy Panel
  • Connect to your WiFi
  • Configure AqualinkD
 

General information not covered in above links TLDR; summary

  • Default user and password are pi and aqualinkd
  • root password is not set, can set with sudo passwd root
  • As with all systems, please change passwords to your liking
  • LED's (can be seen through the vent holes in case)
    • Solid Red - Power up and booted
    • Solid Green - Power up
    • Flashing Red/Green - booting

    Managing Linux Computer

    If you are familiar with basic linux computer, please ignore the below.

    You will need terminal access to the Raspberry Pi Compute Module for time to time to perform tasks. SSH is enabled by default so all you need is an SSH tool.
    • Apple Mac open a terminal and type ssh pi@aqualinkd.local (change aqualinkd.local to IP address if needed)
    • Windows 10 and above open a cmd prompt and type ssh pi@aqualinkd.local (change aqualinkd.local to IP address if needed)
    • All Windows, download a SSH tool like Putty and use ssh pi@aqualinkd.local (change aqualinkd.local to IP address if needed)

    Setting Time

  • The Linux OS will keep perfect time with an online time server (NTP) and AqualinkD will keep the Jandy control panel synced with the Linux OS. ie Automatically updating for daylight savings, and overcoming the terrible internal clock on the Jandy control panel.
  • By default the AqualinD Kit comes with the time zone set to the address the unit was shipped to, if you need to change that, the easiest way is to SSH to AqualinkD unit and set timezone. sudo raspi-config select Localization Options->Timezone->(select time zone)
