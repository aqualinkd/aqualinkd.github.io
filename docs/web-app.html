AqualinkD has a built in webapp so you don't need to use an app store (ie google/apple) to download the app.

Go to aqualinkd's main web page in your mobile browser http://aqualinkd.local(URL will depend on your config).
  • Then use "add to home screen" or "save to desktop" option in the mobile browser.
  • Since AqualinkD's default web interface adheres to all mobile webapp standards, when doing the above your mobile device will create a new AqualinkD app. (The below icon will appear on your device)


AqualinkD webapp has the same functionality as the webpage, except you cannot navigate to the Aqmanager page.

Each pool device (circuit) is shown as a tile.
  • Tiles that support on/off can be pressed quickly to change state
  • Tiles can also be pressed for a few seconds to bring up extra options giving you the ability to set timers, change RPM, SWG, Light options etc
  • Some tiles (ie sensors) like Air & Water Temperature, SWG PPM have no options or on/off
