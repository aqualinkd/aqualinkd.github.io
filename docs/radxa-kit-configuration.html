Complete details are in AqualinkD wiki

  • Once powered on, AqualinkD will be trying to connect to your control panel and self configure

  • In a web browser go to http://aqualinkd.local if that doesn't work, your device/network doesn't support MDNS and you will need to use the machine IP address eg http://ip.address/
    • If you don't know the IP address please go back to Configure WiFi stage and look how to obtain that
       
    • Go to aqmanager by selecting the 3 lines in top left corner


    • In aqmanager, once AqualinkD has finished self configuring, expand the "Config" section and select "edit config"


    • If you have no connection or errors, you may need to check your wiring to the Jandy Panel
      • Toggle the advanced settings, and change serial port appropriately

    • AqualinkD should have auto configured your Panel Type & Size, and appropriate DEvice ID's
      • Please check the panel type is correct. If not see Configuration
    • Check & change values and select "Save" (even if you didn't change anything)
      • The values shown at this point are suggested values, and have not saved to the configuration

    • Please go onto configuring the rest of AqualinkD
      • Configuring Buttons/Circuits should be done next
      • Quick details of all conf options are in the Configuration section
      • Complete configuration details are in the AqualinkD wiki

    • Once configuration is complete, Select Restart AqualinkD
